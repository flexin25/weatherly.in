About Weatherly

A modern, minimalist weather application

What is Weatherly?

Weatherly is a sleek, user-friendly weather application that provides real-time weather information for any city around the globe. Built with a focus on clean design and intuitive user experience, it delivers essential weather data in a beautiful, easy-to-read format.

Key Features

  • Real-Time Weather Data: Get up-to-date weather information powered by OpenWeatherMap API
  • Comprehensive Metrics: View temperature, humidity, atmospheric pressure, and wind speed
  • 5-Day Forecast: Plan your week with extended weather predictions
  • Air Quality Index: Monitor real-time AQI levels and air pollution data for any city
  • Pollutants Tracking: Track detailed pollutant levels including PM2.5, PM10, CO, NO₂, O₃, and SO₂
  • Search Any City: Access weather and air quality data for cities worldwide
  • Responsive Design: Seamlessly works across all devices
  • Glassmorphic UI: Modern, blurred-glass interface design

Technology Stack

Weatherly is built using vanilla HTML, CSS, and JavaScript, demonstrating that powerful applications don't always need heavy frameworks. The app leverages modern CSS features like backdrop-filter for the glassmorphic design and the Fetch API for seamless data retrieval.
